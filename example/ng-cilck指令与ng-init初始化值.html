小结

1. Angular
MV*——数据、表现分离
减轻程序员的负担，减少重复劳动
扩展了HTMl的功能：颠覆性、可扩展
2. 基本使用
MVC
M : Model 模型——数据
V : View 视图——表现层
C : Controller 控制器——业务逻辑
ng-app="" : Angular的范围
两个词
双向绑定
依赖注入
饭店：
函数——参数由调用方决定
依赖注入——参数由定义方决定
重点
1. angular和JavaScript不互通
2. Angular的开发模式和传统开发模式不同——只需要顶住数据
3. 数据是核心 ng-开头
Angular接管了UI
Angular 与JavaScript互通 需要经过Controller
AngularJs；MVx框架 把数据、表现、逻辑分离开
方便程序员——避免重复劳动（获取、事件）
ng-model 数据源 双向绑定
ng-bind 输出数据
